# Backend
1. Ativar ambiente
```bash
python -m venv venv
```

- Windows:
```bash
   cd backend\venv\Scripts\activate
```

- Linux/MacOs:
```bash
   cd backend/venv/Scripts/activate
```

2. Instalar requisitos
```bash
   pip install -r requirements.txt
```

3. Executar testes
```bash
    python -m unittest discover
```

4. Iniciar aplicação
Ir até backend/VotesCSV e executar:
```bash
   python manage.py runserver
```

# Frontend
1. Instalar pacotes
```bash
   npm install
```

2. Iniciar aplicação
```bash
   npm run start
```
